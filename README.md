This project implements three core numerical analysis tasks:
1. **Interpolation**: Lagrange Polynomial & Cubic Spline
2. **Data Analysis**: Algorithm performance comparison with Excel data
3. **Numerical Integration**: Trapezoidal, Simpson, and Gaussian Quadrature
